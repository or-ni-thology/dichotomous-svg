# Dichotomy

*An inverse key by SVG to the Scolopacidae of the North Pennines*

**Live:** [or-ni-thology.cloud](https://or-ni-thology.cloud/dichotomous-svg) · **Licence:** [CC0 1.0](LICENSE) — public domain, take it, change it, no credit needed

---

### WHY

A dichotomous key is a ladder of either/ors that a naturalist climbs down until one bird remains. This instrument turns the ladder over. Rather than reading a wader down to a name, the user builds a wader up from its characters and reads off the name it cannot help but be.
Set up to describe the Scolopacidae of the North Pennines,the concept could in theory work for the Threskiornithidae of Los Llanos with some minor adjustments to bill curvature. 

The tool is *very useful* for creating images for games of Scolopictionary.
